# Colorized Grep
> Minor tool to colorize terminal output

## Simple color grep

![demo](demo.gif)

* -p or --pattern, using the format **pattern:color** to colorize the string match

## Piped input

![demo-stdinput](demo-stdinput.gif)

* -i or --input, get input from standard input (piped)
